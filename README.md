TaskFlow
TaskFlow is a simple, intuitive web app for managing your to-do tasks. It lets you add, edit, and organize tasks easily, helping you keep track of what needs to be done.

Key Features

Task Management: Add, edit, and delete tasks through an easy-to-use interface.

LocalStorage: All tasks are stored in your browser, so your data persists even after you close the app.

Team View: Organize and view tasks by team or collaborator for easy group project management.

Dark Mode: Toggle between light and dark themes for a comfortable viewing experience in any lighting.


Technology Stack
React: Front-end library for building the user interface.
TypeScript: Strongly-typed language for writing safer JavaScript.
Tailwind CSS: Utility-first CSS framework for styling.
LocalStorage: Browser API for persistent, client-side data storage.

Setup Instructions
Clone the repository:
git clone https://github.com/HetPatel2412/taskflow.git

Navigate into the project directory:

cd taskflow

Install the dependencies:

npm install

Start the development server:

npm run dev

Open http://localhost:3000 in your browser to view the app.

Deployment

TaskFlow can be easily deployed on Vercel. After pushing your code to a GitHub repository, import the project in Vercel and it will handle building and hosting automatically.
